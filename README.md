# مُذكّري الذكي (Smart Reminder AI)

تطبيق إدارة مهام ذكي مدعوم بالذكاء الاصطناعي بواجهة عربية كاملة (RTL).

## ✨ المميزات

- 🔐 **نظام مصادقة كامل** — تسجيل حساب، دخول، خروج (NextAuth.js)
- 🧠 **ذكاء اصطناعي** — اقتراح أوقات مثالية لإنجاز المهام + رؤى تحليلية
- 📊 **إحصائيات تفصيلية** — رسوم بيانية للإنجاز والفئات والأولويات
- 📱 **PWA** — قابل للتثبيت كتطبيق جوال
- 🌗 **وضع ليلي/نهاري** — دعم كامل للثيمين
- 🛡️ **حماية الملكية** — رخصة CC BY-NC-ND 4.0
